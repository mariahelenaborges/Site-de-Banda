2. Box Model 
Usei todos (margin + border + padding + content) 

3. Background 
Escolhi adicionar uma imagem de fundo com "background-image" e utilizei o "cover" para que a imagem ocupasse toda a tela e "center" para centralizá-la. 

4. Cores 
Utilizei uma ou duas a mais, mas foi predominantemente preto e vermelho. Também usei o "rgba" com transparência de cor preta para o destaque. 

5. Layout 
Usei o Flexbox para a imagem e texto ficarem lado a lado e o alinhamento visualmente agradável. 

6. Imagens 
Padronizei tamanho e manti o formato quadricular de cada foto dos integrantes com box model.

7.  Lista 
Eu fiz a lista e usei o seguinte código para organizar os pontinhos dentro da caixa: 
ul {
    list-style-position: inside;
}

8. Tabela 
Eu fiz a tabela e estilizei a tabela também usando o css integrado ao meu html. Juntei as bordas e organizei o conteúdo dentro.

Por fim, utilizei o "text-align: center" para centralização e "font-size: 50px" para a tipografia ajustada.
